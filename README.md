## Epsilon (development name)
Goal: A fast and robust visualization and simulation tool.

Components:
- **Visualization Engine**: A high-performance OpenGL-based rendering engine for real-time graphics.
- **Code Processing**: Epsilon integrates the V8 JavaScript engine to handle complex computations efficiently, with the benefits of a widely-used language.
- **Editor**: A user-friendly interface with tile-able panels, syntax highlighting, and flexibility.
